# The Will of the City

A project inspired by one the Five Finger, one of the most powerful Syndicate in the City that can rival a Wing, The Index, by the iconic world of Project Moon. 
The Index, symbolize one of the many nature of human, which is faith, a faith that is based on the will of the city.

# Project

This is a project that I am working on, with Angular 2 version 21.2.7, to run the application use :
```
ng serve
```
Once the server is running, open a browser and navigate to http://localhost:4200/. The application will automatically reload whenever you modify any of the source files.
